---
description: How to set up the LingoFlow project in a new Opencode workspace
---

1. Ensure the project is in the `Opencode` workspace.
2. Open the folder `C:\Users\famil\Documents\Opencode\Lingoflow` in Opencode.
3. Run the following command in the terminal:
   ```bash
   npm install && npm run dev
   ```
4. Verify that the `.env.local` file contains the correct `VITE_GEMINI_API_KEY`.
5. Access the AI Tutor and verify connectivity.
